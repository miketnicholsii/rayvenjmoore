# Rayven J. Moore, CPA

Personal website for Rayven J. Moore, CPA - Converting Complexity Into Clarity.

## About

A professional portfolio and personal brand website showcasing expertise in financial education, wealth building, and community impact.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
